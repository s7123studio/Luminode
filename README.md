## ✨ 核心特性

- **极简内核** - 仅 150KB 核心代码，零外部依赖
- **光速响应** - 内置OPcache优化，毫秒级路由匹配
- **模块化架构** - 按需加载组件，自由搭配功能
- **安全加固** - 自动CSRF防护 + XSS过滤 + SQL注入防御
- **全栈支持** - 从路由到模板引擎一站式解决方案
